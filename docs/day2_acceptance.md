# Day 2 Acceptance - Source Connectivity Probe

## Today objective
- Build a minimal, executable source connectivity probe for Macau court judgment search.
- Collect concrete evidence to decide whether Day 3 implementation should remain on `requests` or escalate to Playwright.

## Deliverables
- Probe script: `crawler/source_probes/court_judgment_search_probe.py`
- Probe-stage source spec: `crawler/parsers/court_judgment_search_spec.md`
- Probe artifacts generated by script:
  - `data/raw/court_probe/researchjudgments.html`
  - `data/raw/court_probe/researchjudgments_text_preview.txt`
